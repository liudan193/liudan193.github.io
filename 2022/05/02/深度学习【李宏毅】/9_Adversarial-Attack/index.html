Attack的例子

这里有一张示例图片，经过一个Network进行分类，告诉你是个什么类别的东西

现在要做的就是，在这个照片上面加一个非常小的杂讯，一张照片可以被看作是一个非常长的向量,我们在这个非常长的向量上加入,每一个维度都加入一个小小的杂讯，小到人肉眼分辨不出来

有被加杂讯的照片叫做 Attacked Image，有被攻击的照片，那还没有被加杂讯的照片呢，我们一般就叫做 Benign Image，它是好的 Image，它是还没有被攻击的图片

将 Attacked Image 丢入 Network 内，让它的输出不可以是猫,要变成其他的东西

攻击大致上可以分成两种类型，一种是没有目标的攻击，没有目标的攻击是，原来的答案是猫，只要你能够让 Network 的输出不是猫，你就算是成功了；还有另外一种更困难的攻击，是有目标的攻击，也就是说我们希望 Network，不止它输出不能是猫，还要输出别的东西，比如说 我们希望加了一个杂讯以后，Network 输出呢是海星，它把猫错误判断成一隻海星，才算是攻击成功

Attack训练的方法

1. 目标和损失函数

没有目标的攻击

假设 Network 的参数是固定的，现在目标就是找到一个新的图片，希望 y 跟 ŷ 的差距越大越好，y 为本来应该的输出，ŷ 为想要输出的错误输出

损失函数：由于我们希望 y 跟 ŷ 的差距越大越好，因此我们可以取一个符号，在 Cross Entropy 前面加一个负号，作为我们的损失函数：

$$
L\left( x \right) \ =\ -e\left( y,\hat{y} \right)
$$

有目标的攻击

那我们会先设定好我们的目标，我们用 $y^{target}$ 来代表我们的目标，那 ŷ 其实是一个 One-Hot Vector，$y^{target}$ 也是一个 One-Hot Vector，那我们现在希望这个 y 不止跟 ŷ 越远越好，我们还要跟 $y^{target}$ 越近越好

损失函数：

$$
L\left( x \right) \ =\ -e\left( y,\hat{y} \right) \ +\ e\left( y,y^{target} \right)
$$

2. 使人眼感知不到

增加约束

光是找一个 x，去 Minimize Loss 是不够的，因为我们其实还期待说，我们加入的杂讯越小越好，也就是我们新找到的图片，可以欺骗过 Network 的图片，跟原来的图片要越相近越好，x 跟 x0 要越近越好

所以我们在 Optimization 的时候，我们还会多加入一个限制，这个限制是：

$$
d\left( x^0,x \right) \ \le \ \varepsilon
$$

计算距离

那么怎么计算这个距离 $d$ 呢？

有各式各样不同的算法，那了等一下符号的方便起见呢，我们假设 $x$ 是一个向量，因为它是个图片 所以它是个向量嘛，$x^0$是另外一张图片，它也是一个向量，这两个向量相减 我们叫它 $Δx$

那这个距离

  • 你可以定 L2-Norm 当做它们的距离，L2-Norm 是所有 $\Delta x$ 的平方之和，也可以开个根号
  • 另外还有一个定义是 L-Infinity，L-Infinity 是所有 $|\Delta x|$ 中最大的那个

选择距离

那有各种不同的方法,可以计算两张图片之间的距离,但是我们在决定要使用哪一种方法,来计算图片的距离的时候,其实我们应该把人类的感知把它考虑进来，比较接近人类的感知能力的应该是 L-Infinity

在具体的问题中，需要根据人类的感知来定制这个 distance，以确保不被人类发觉，这个部分就要用到 Domain Knowledge

3. 优化

除了 distance 的限制之外，这跟我们 Train 一个模型其实没有什么差别，这次只是把参数改成 Network 的 Input 而已，把 Input 那一张 Image,看作是 Network 参数的一部分,然后 Minimize 你的 Loss Function 就结束了

但是还是需要调整一下 Gradient Descent，首先需要一个初始值，选择 $x^0$ 正好解决了小 distance 的问题

接下来我们得把 Constraint(约束) 加进去，把 $x^t$ 做个修改,把它改回符合限制就结束了（具体而言，在可行域里面找一个跟超出的点最近的位置，把这个点拉进来就行了）
