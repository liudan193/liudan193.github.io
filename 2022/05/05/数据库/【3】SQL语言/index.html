SQL分类

  • DDL（Data Definition Language）语句
    创建数据库、表等操作
  • DML（Data Manipulation Language）语句
    操作数据库数据（增删改查等操作）
  • DQL（Data Query Language）语句
    查询
  • DCL（Data Control Language）语句
    创建、控制用户以及权限等

数据类型

分类 类型 大小 有符号范围 无符号范围 描述
数值 int 4 bytes 整数值
数值 float 4 bytes 单精度浮点数
数值 double 8 bytes 双精度浮点数
字符串 char 0-255 bytes 定长字符串
字符串 varchar 0-65535 bytes 变长字符串
日期 pass

varchar(20)表示的是varchar数据的数据长度最大是20，超过则数据库不会存储；

条件符号

条件符号 作用
> 大于
< 小于
>= 大于等于
<= 小于等于
= 等于
<> 或 != 不等于
between… and… 在某个范围之内（包含最大值和最小值）
in(…) 在后面的列表中的值，多选一
like 占位符 模糊匹配，_匹配单个字符，%匹配多个字符
is null 是NULL
and 或 && 与
or 或 || 或
not 或 ! 非
