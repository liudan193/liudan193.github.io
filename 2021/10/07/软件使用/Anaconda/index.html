Anaconda使用教程

发表于2021-10-07|更新于2021-09-21|软件使用
|阅读量:

软件介绍

Anaconda 就是可以便捷获取包且对包能够进行管理，同时对环境可以统一管理的发行版本。Anaconda包含了conda、Python在内的超过180个科学包及其依赖项。

使用conda来方便地在不同环境之间进行切换，更简单地管理Python的虚拟环境。

Anaconda有图形界面和命令行两种方式，由于图形界面运行较慢，因此常使用命令行的方式进行虚拟环境的管理。

使用教程 - 指令

命令行界面 - Windows用户打开“Anaconda Prompt”进行操作，macOS和Linux用户打开“Terminal”（“终端”）进行操作。

管理软件

  1. 显示当前安装conda的版本号
  2. 更新conda至最新版本
  3. 查看conda帮助信息
  4. 卸载conda

① Linux 或 macOS

即删除Anaconda的安装目录。根据安装的Anaconda版本选择相应的卸载命令。

② Windows
控制面板 → 添加或删除程序 → 选择“Python X.X (Anaconda)” → 点击“删除程序”

  1. 换源

建议进图形界面换源
依次点击 Channels -> Add -> 输入源 -> 敲击Enter
重复三次该操作,分别输入

点击右下角的Update channels

管理环境

以下均为Windows操作

  1. 创建新环境
  2. 切换环境
  3. 退出环境
  4. 列出已经创建的环境
  5. 复制环境
  6. 删除环境

管理包

  1. 安装包

还有一种可能,conda无法安装包,需要使用pip安装

如果想在指定环境中使用pip进行安装包，则需要先切换到指定环境中，再使用pip命令安装包。

  1. 列出已经安装的包

  2. 精确查找可用包

  3. 模糊查找

    对于一些不熟悉的包,也可以直接到图形界面搜索

  4. 卸载包

  5. 更新包
