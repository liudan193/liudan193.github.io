MIPS指令的详解

本文在阅读了《计算机组成与设计：硬件/软件接口》第二章节，了解一些MIPS指令的基础上阅读。

32个寄存器（register）

MIPS有32个通用寄存器（$0-$31），某个寄存器本身并没有指定的功能，但是在实际使用中，为了方便和统一，每个寄存器都根据约定被赋予了指定的功能。

在使用中，既可以使用数字编号，也可以使用约定好的符号（也是建议的用法）。

32个寄存器的数字编号、符号、功能的对应关系如下：

编号 符号 功能
$0 $zero 常量0(constant value 0)
$1 $at 保留给汇编器(Reserved for assembler)的，没啥用
$2-$3 $v0-$v1 函数调用返回值(values for results and expression evaluation)
$4-$7 $a0-$a3 函数调用参数(arguments)
$8-$15 $t0-$t7 暂时的(temporary)
$16-$23 $s0-$s7 保存的(或如果用，需要SAVE/RESTORE的)(saved)
$24-$25 $t8-$t9 暂时的(temporary)
$28 $gp 全局指针(Global Pointer)
$29 $sp 堆栈指针(Stack Pointer)
$30 $fp 帧指针(Frame Pointer)
$31 $ra 返回地址(return address)

32位立即数

因为有时候立即数会大于16位，但是I型指令的立即数位数只有16位，因此有了32位立即数的操作。

MIPS为32位立即数专门设置了lui指令，用于读取立即数的高十六位，低十六位全为0，再组合其他命令（比如ori）设置剩下的低十六位。

例如：将下面的32位常量存到寄存器$s0中
0000 0000 0011 1101 000 1001 0000 0000

lui $s0, 61
# 执行该语句之后，$s0内的值为：
# 0000 0000 0011 1101 0000 0000 0000 0000

ori $s0, $s0, 2304
# 通过执行或操作，将剩下的低16位转换为应该的数值
# 经过或操作之后，$s0的值为：
# 0000 0000 0011 1101 000 1001 0000 0000

指令

在MIPS中，指令分为三类，即三种指令格式，分别为R(register 寄存器)、I(immediate 立即数)、J(jump 跳转命令)三种类型。

每种指令格式均为32bit的。

R型 - 用于寄存器

字段 opcode rs rt rd shamt funct
占位(bit) 6 5 5 5 5 6

每个字段的解释：

  • opcode: 操作码，所有R型指令都全为0
  • rs：寄存器编号，对应第一个源操作数（registrer source）
  • rt：寄存器编号，对应第二个源操作数（register target）
  • rd：寄存器编号，对应结果（register destination）
  • shamt：偏移量，只在移位指令中使用，如sll、srl（shift amount）
  • funct：功能码，指定指令的具体功能，如add、sub（function）

算术运算

指令 功能
add $s1,$s2,$3 $s1 = $2 + $3
sub $s1,$s2,$3 $s1 = $2 - $3

逻辑运算

指令 功能
and $s1,$s2,$3 $s1 = $2 & $3
or $s1,$s2,$3 $s1 = $2 | $3
nor $s1,$s2,$3 $s1 = ~($2 | $3)（或非，不是非）

移位指令（R特有，I没有）

指令 功能
sll $s1,$s2,10 $s1 = $2 << 10（shift left logical）
srl $s1,$s2,10 $s1 = $2 >> 10

解释：
无符号的二进制数左移一位相当于将该数乘以2，左移三位相当于乘以8；类比10进制，左移一位相当于乘以10，两位相当于乘以100。

判断大小

指令 功能
slt $s1,$s2,$s3 如果 $s2<$s3, $s1 为 1; 否则为 0 （set on less than）

跳转

指令 功能
jr $ra go to $ra

I型 - 用于立即数

字段 opcode rs rt immdiate
占位(bit) 6 5 5 16

每个字段的解释：

  • opcode: 操作码
  • rs：第一个源操作数，是寄存器操作数（registrer source）
  • rt：目的寄存器编号，用来保存运算结果（register target）
  • immediate：第二个源操作数，立即数

存字、取字（I特有，R没有）

指令 功能
lw $s1,20($s2) $s1 = Memory[$s2 + 20]（load word）
sw $s1,20($s2) Memory[$s2 + 20] = $s1（store word）

决策指令（I特有，R没有）

指令 功能
beq $s1,$s2,25 如果$s1 == $s2，go to PC+4+4*25（binary equal）
bne $s1,$s2,26 如果$s1 != $s2，go to PC+4+4*26（binary not equal）

算术运算

指令 功能
addi $s1,$s2,20 $s1 = $2 + 20

逻辑运算

指令 功能
andi $s1,$s2,20 $s1 = $2 & 20
ori $s1,$s2,20 $s1 = $2 | 20

判断大小

指令 功能
slti $s1,$s2,20 如果 $s2<20, $s1 为 1; 否则为 0 （set on less than）

J型 - 用于跳转指令

字段 opcode immdiate
占位(bit) 6 26

每个字段的解释：

  • opcode: 操作码
  • immediate：立即数

J型指令

指令 功能
j 2500 goto 4*2500
jal 2500 $ra = PC + 4，go to 4*2500

常用指令的机器码

指令 格式 opcode funct
add R 0 32
sub R 0 34
sll R 0 0
addi I 8 Nan
lw I 35 Nan
sw I 43 Nan

注：

  1. 以上均为10进制数字
  2. R型指令的 opcode域 均为 0
