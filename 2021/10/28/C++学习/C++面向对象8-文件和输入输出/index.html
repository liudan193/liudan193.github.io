1. C++的输入输出系统

1.1 C++流的概念

C++用流来作为输入输出的抽象，C++中用iostream库中定义了标准的输入输出类库。

  • 头文件
头文件 作用
iostream 输入输出操作的基本信息，如cin、cout
fstream 用于管理文件的I/O操作
strstream 用于字符串的I/O操作
iomanip 用于输入输出的格式控制，如setw、fixed
  • 定义的流class
类名 作用 头文件
ios 流基类 iostream
istream 通用的流输入类 iostream
ostream 通用的流输出类 iostream
iostream 通用的流输入输出类 iostream
ifstream 文件流输入类 fstream
ofstream 文件流输出类 fstream
fstream 文件流输入输出类 fstream
istrstream 字符串流输入类 strstream
ostrstream 字符串流输出类 strstream
strstream 字符串流输入输出类 strstream

1.2 预定义的对象及其成员对象

cin： 标准输出设备
cout： 标准输出设备
cerr： 标准错误输出设备
clog： 标准错误输出设备

其中：cerr输出是不经过缓冲区的，直接输出到显示器中；而clog是经过缓冲区的。

输入输出有一些成员函数：

cout.put(char c)：输出一个字符；
cin.get(ch)：从输入流中读入一个字符，包括空白字符，输入EOF信号时会失败，返回0；
cin.getline(字符指针，字符个数n，中止标志字符)：读入一行文字；
cin.ignore(10,'t')：跳过流中读入10个字符，碰到 ‘t’ 字符时停止跳跃（cin.ignore()跳过一个字符，默认参数为1，EOF）；

2. 文件流

文件分为文本文件和二进制文件两种。

1. 文件流对象：

对象 说明
fstream fstrm 创建一个未和任何文件绑定的文件流对象
fstream fstrm(s) 初始化创建一个和文件 s 绑定的文件流对象
fstream fstrm(s, mode) 初始化创建一个和文件 s 绑定的文件流对象，打开方式为 mode

2. 文件流对象的成员函数：

成员函数 说明
fstrm.open(s) 绑定文件 s，即打开文件 s
fstrm.open(s, mode) 以打开方式 mode 绑定文件 s，即打开文件 s
fstrm.close() 取消绑定，即关闭文件
fstrm.is_open() 判断绑定的文件是否打开成功

3. 打开方式：

mode 说明
ios::in 读文件
ios::out 写文件
ios::binary 打开二进制文件
ios::app 文件尾部打开
ios::ate 所有输出附加在文件末尾
ios::trunc 如果存在该文件则先删除该文件
ios::nocreate 不建立文件，不存在则打开失败
ios::noreplace 不覆盖文件，存在则打开失败

3. 字符串流

以内存中的字符串类对象或者字符数组为输入输出对象的数据流，也叫内存流

涉及头文件sstream，定义三种类型：istringstream（读string）、ostringstream（写string）、stringstream（读写string）

常见语句 说明
sstream strm 创建一个未和任何string类型绑定的stringstream对象
sstream strm(s) 创建一个sstream流，该流与s的拷贝绑定
strm.str() 返回与strm流绑定的string
strm.str(s) 拷贝string s到流strm，返回void
