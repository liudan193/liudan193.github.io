加减乘除运算

1. 加减法

  • 加法
    从左到右一次计算即可

  • 减法
    如 7 - 6，即为 7 + [6]补码

  • 溢出

A+B <0 <0 >=0

2. 乘法

采取移位相加的方法

以 0010 * 0011 为例：

  1. 初始时
    被乘数：0000 0010
    乘数：             0011
    积 ：       0000 0000

  2. 取乘数末位

    乘数末位 操作
    0 pass
    1 将被乘数加到积里面

  3. 被乘数左移1位
    乘数右移1位

解释：为什么题目中4位的被乘数成为了8位？
因为在硬件设计中，整数是32位存储的，在乘法器硬件的设计中，为了精度，被乘数和积被被放在了64位的寄存器中，因此该例子以4位存储为例，因此被乘数和积都是8位的。

3. 除法

以 0111 ÷ 0010 为例：

  1. 初始值：余数 = 被除数，商 = 0000；
  2. 余数 = 余数 - 除数；
  3. if 余数 ≥ 0: 商左移，最低位设1，
    if 余数 < 0: 余数 = 余数 + 除数，商右移，最低位设0；
  4. 除数右移，回到第 1 步，直到第五次完成（32位寄存器为第33次结束的时候）。
